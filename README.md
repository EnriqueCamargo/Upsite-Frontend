 [ Red Social Académica para el Futuro ]


🎯 Descripción del Proyecto

UPSITE no es solo otro foro. Es un ecosistema de comunicación centralizado diseñado específicamente para cerrar la brecha entre el personal docente y la directiva de una institución.

Esta plataforma funciona como una red social académica que trasciende las limitaciones de las herramientas de mensajería instantánea convencionales. Nuestro objetivo principal es eliminar el ruido, asegurando que la información crítica llegue a quien debe llegar, sin perderse en hilos interminables de conversación.
✨ Características Principales

    🚀 Publicaciones Directivas: Un canal dedicado para que los directivos publiquen avisos oficiales y directrices.

    📣 Tablón de Anuncios Académico: Espacio centralizado para que los profesores compartan información relevante, recursos o soliciten colaboración.

    ✉️ Sistema de Notificaciones por Correo: Se integra con el servidor de correo para notificar automáticamente a los usuarios sobre nuevas publicaciones críticas o actualizaciones.

    🔄 Persistencia de Información: La información importante está categorizada y es buscable, garantizando su disponibilidad a largo plazo.

    🔒 Accesibilidad Asegurada: Garantiza que todos los miembros de la red puedan ver la información, eliminando la exclusión accidental de grupos.

🛠️ Stack Tecnológico

Este proyecto ha sido desarrollado utilizando tecnologías de vanguardia para asegurar el rendimiento, la escalabilidad y la robustez.

    ☕ Java 25: Implementando las últimas características del lenguaje para un backend moderno y eficiente.

    🍃 Spring Boot 3.x: El framework de backend más potente para microservicios y aplicaciones web.

    🐘 PostgreSQL: Una base de datos relacional potente y confiable para la gestión segura de los datos.

    📦 Maven: Herramienta de gestión de dependencias y construcción.

    🛡️ Spring Security: Para una gestión de usuarios y roles segura y granular.

📋 Prerrequisitos para el Despliegue

Antes de comenzar, asegúrate de tener instalado:

    Java Development Kit (JDK) 25 o superior.

    PostgreSQL (Servidor en ejecución).

    Maven (Instalado y configurado en el PATH).
